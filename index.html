Set Up OpenClaw on a VPS

~10 minutes · 6 steps · your own AI assistant

1. Provision a Server

Create a server on Hetzner Cloud:

  • OS: Ubuntu 24.04
  • Type: CX22 (2 vCPU, 4GB RAM)
  • SSH Key: Add yours during setup

SSH in as root:

ssh root@YOUR_SERVER_IP

2. Create a User

Create a dedicated user with passwordless sudo:

adduser claw
usermod -aG sudo claw

# Passwordless sudo
echo "claw ALL=(ALL) NOPASSWD:ALL" \
  > /etc/sudoers.d/claw
chmod 440 /etc/sudoers.d/claw

Copy your SSH key to the new user:

mkdir -p /home/claw/.ssh
cp ~/.ssh/authorized_keys /home/claw/.ssh/
chown -R claw:claw /home/claw/.ssh

Disconnect and reconnect as claw:

ssh claw@YOUR_SERVER_IP

3. Install Homebrew

/bin/bash -c "$(curl -fsSL \
  https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Add to your PATH:

echo 'eval "$(/home/linuxbrew/.linuxbrew/bin/brew shellenv)"' \
  >> ~/.bashrc
eval "$(/home/linuxbrew/.linuxbrew/bin/brew shellenv)"

4. Install OpenClaw

brew install openclaw

Run the setup wizard:

openclaw onboard

This walks you through choosing a provider, entering your API key, and connecting a chat channel.

5. Add Your API Key

Get an API key from Anthropic (recommended). The wizard asks for it, or configure later:

openclaw configure --section auth

6. Start

openclaw gateway start --daemon

Done. Send it a message on Telegram or Slack — it'll respond.

Quick Reference

CommandDescription

openclaw gateway start --daemonStart in background
