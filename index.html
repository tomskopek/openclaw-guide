Set up your own AI assistant on a VPS

A minimal, step-by-step guide to get OpenClaw running on a fresh Hetzner server. Six steps, no fluff.

üí∞ Total cost: ~$10/month
Step 1

Provision a Server

Go to Hetzner Cloud and create a new server:

  • OS: Ubuntu 24.04
  • Type: CX22 (2 vCPU, 4GB RAM) ‚Äî plenty to start
  • Location: Closest to you
  • SSH Key: Add your public key during setup

Once it's live, grab the IP from the Hetzner dashboard and SSH in:

# Connect as root
ssh root@YOUR_SERVER_IPterminal
Step 2

Create a User

Don't run OpenClaw as root. Create a dedicated user with passwordless sudo:

# Create user
adduser claw
usermod -aG sudo claw

# Passwordless sudo (so OpenClaw can install packages)
echo "claw ALL=(ALL) NOPASSWD:ALL" > /etc/sudoers.d/claw
chmod 440 /etc/sudoers.d/clawbash

Copy your SSH key to the new user, then switch over:

# Copy SSH key from root to claw
mkdir -p /home/claw/.ssh
cp ~/.ssh/authorized_keys /home/claw/.ssh/
chown -R claw:claw /home/claw/.ssh

# Now disconnect and reconnect as claw
ssh claw@YOUR_SERVER_IPbash
Step 3

Install Homebrew

OpenClaw installs via Homebrew. Run the installer:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"bash

Then add it to your PATH (the installer will show these exact lines):

echo 'eval "$(/home/linuxbrew/.linuxbrew/bin/brew shellenv)"' >> ~/.bashrc
eval "$(/home/linuxbrew/.linuxbrew/bin/brew shellenv)"bash
Step 4

Install OpenClaw

brew install openclawbash

Then run the setup wizard:

openclaw onboardbash
üí°The wizard walks you through choosing a model provider, entering your API key, and connecting a chat channel (Telegram, Slack, etc.).
Step 5

Add Your API Key

You need an API key from a model provider. Anthropic (Claude) is recommended.

The onboard wizard asks for it automatically. To update it later:

openclaw configure --section authbash
Step 6

Start the Gateway

# Start in the background (persists after disconnect)
openclaw gateway start --daemonbash

That's it. OpenClaw is running. If you connected Telegram or Slack during setup, send it a message ‚Äî it'll respond.

Check status anytime:

openclaw statusbash
